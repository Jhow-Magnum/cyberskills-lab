Treinamento Prático em Cibersegurança

Bem-vindo ao CYBERSKILLS LAB

Esta plataforma nasceu da comunidade, para a comunidade. Criada para quem quer aprender cibersegurança na prática, sem barreiras, sem custos, sem complicações.

Sabemos que nem todos têm acesso a máquinas com dual boot, ambientes Linux configurados ou recursos para plataformas pagas. Por isso, o CYBERSKILLS LAB roda 100% local, no seu computador, usando Docker para isolar cada desafio de forma segura.

Para quem é este projeto?

  • Estudantes de segurança da informação que querem praticar
  • Entusiastas de Linux e sistemas operacionais
  • Profissionais que se preparam para certificações (OSCP, CEH, etc)
  • Professores que buscam material prático para suas aulas
  • Qualquer pessoa curiosa sobre como a segurança funciona na prática

Por que Open Source?

Porque conhecimento deve ser livre. O código está aberto no GitHub para que você possa:

  • Estudar como a plataforma foi construída
  • Adicionar seus próprios desafios e categorias
  • Corrigir bugs e melhorar funcionalidades
  • Adaptar para suas necessidades específicas
  • Aprender com o código de outros desenvolvedores

Esta é a primeira versão

Sim, ainda há muito a melhorar. Mas é assim que projetos open-source crescem: com a colaboração da comunidade.

Se você tem ideias, encontrou bugs, ou quer adicionar novos desafios, sua contribuição é bem-vinda. Não precisa ser um expert, todos começamos de algum lugar.

O que você vai encontrar aqui

  • 40 desafios práticos em 6 categorias diferentes
  • Ambientes Docker isolados para cada lab
  • Terminal web integrado (sem precisar SSH externo)
  • Sistema de pontuação e ranking
  • Hints progressivos para quando você travar
  • Writeups automáticos após completar os desafios

Escolha um laboratório no menu ao lado e comece sua jornada.

Lembre-se: a melhor forma de aprender é fazendo. Boa sorte, hacker!

💡 Dica: Instale como Aplicativo

Você pode instalar o CYBERSKILLS LAB como um aplicativo nativo no seu computador. Ele abrirá em janela própria, sem a barra do navegador, proporcionando uma experiência mais imersiva.

Como instalar:

  • Chrome/Edge/Brave: Acesse a plataforma, procure o ícone de computador com seta (⊕) no canto direito da barra de endereço (ao lado da estrela de favoritos) e clique em "Instalar"
  • Firefox: Clique nos três traços (☰) no canto superior direito, depois em "Instalar este site como aplicativo"
  • Safari (macOS): Clique no botão de compartilhar (quadrado com seta para cima) e selecione "Adicionar à Dock"

⚠️ Importante: O aplicativo só funcionará se a plataforma estiver rodando no terminal (bash start.sh). Sem o servidor ativo, o app não abrirá.
